Sprint 3/4/5 Wireframe Blueprint With Specs

Low-fidelity mobile screens for the higher product vision. Sprint 3 is the implementation target. Sprint 4 and Sprint 5 are shown only to validate navigation, future components, and information hierarchy. Measurements are implementation targets in dp/sp for Compose Multiplatform.

Dashed zones = layout regions Solid cards = reusable components Blue outline = Sprint 3 focus Gray = future sprint support

Layout Tokens

  • Phone target: 390 x 844 reference frame; content must adapt to smaller Android widths.
  • Screen padding: 16dp horizontal, 12dp top, 16dp bottom above navigation.
  • Vertical gaps: 8dp compact, 12dp default, 16dp section, 24dp major separation.
  • Radius: 8dp controls, 12dp cards, 16dp bottom nav, 24dp modal sheet top corners.

Type Scale

  • Schedule title: 24sp, semibold/bold, one line before wrapping.
  • Section title: 16sp, semibold.
  • Race title: 16sp, semibold; track/name metadata 13sp.
  • Chip/nav/meta: 12sp; captions and update time 12sp.

Interaction Targets

  • Icon buttons: 44 x 44dp minimum touch target; visual radius 12dp.
  • Chips: 32dp minimum height, 12dp horizontal padding.
  • Race cards: whole card tappable from Sprint 5; minimum internal row target 48dp.
  • Bottom nav: 64-72dp height; each item at least 48dp touch height.
